Sobre:


Na página você encontrará coisas sobre quais problemas aconteceram no meio da pandemia que talvez não tenha notado mas aconteceu, nele terão textos acompanhados com uma imagem para expressar o que foi dito em cada assunto.


Noticias:


Na página aparecerão imagens de algumas noticias tendo links para te direcionar ao site, para que você consiga ver elas completas.

Eu excolhi esse tema para mostrar para as pessoas que se preocupam tanto com o covid-19, que não é errado se preocupar, mas não se preocupam com outras coisas ou até mesmo não notam que estão se machucando. É mais um aviso para que as pessoas procurem se preocupar com as coisas, mas não se esquecer do que mais importa que é a Saúde em todos os sentidos.
